## Props
- `checkIcon` (optional): It accepts an IconPlaceholder component to be displayed inside the checkbox when the value is true.
- `checkboxOnLeft` (optional): A boolean value indicating whether the checkbox should be positioned on the left side.
- `value` (required): A boolean value representing the current state of the checkbox.

## Example Usage
```jsx
import { Checkbox } from '@codeleap/web';

function App() {
  const [checked, setChecked] = useState(false);

  const handleCheckboxChange = () => {
    setChecked(!checked);
  };

  return (
    <Checkbox
      checkIcon={<IconPlaceholder />}
      checkboxOnLeft={true}
      value={checked}
      onChange={handleCheckboxChange}
    />
  );
}
```
