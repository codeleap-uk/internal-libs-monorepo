## Props

- `icon` (string, optional): The name of the icon to be displayed. This prop is optional and can be used to override the `name` prop of the Icon component.
- `name` (string, optional): The name of the icon to be displayed. This prop is optional and can be used to specify the icon using the IconProps type.
