# IconButton Component

## Description
The IconButton component is a button that displays an icon. It allows users to interact with it by clicking or tapping on it.

## Props
| Name     | Type             | Optional | Description                                      |
|----------|------------------|----------|--------------------------------------------------|
| icon     | IconProps['name'] | Yes      | The name of the icon to be displayed.            |
| name     | IconProps['name'] | Yes      | An alternative name for the icon to be displayed.|

## Example Usage
```jsx
import { IconButton } from '@codeleap/mobile';

function MyComponent() {
  return (
    <IconButton icon="heart" />
  );
}
```
